# 컴퓨터 네트워크 HW#1 - Socket Programming

by 심규민 2009-11744

## 빌드

`make`

## 실행법

* `./server <port>`
* `./client <hostname> <port> <window_size> <ack_delay_ms>`

## client 사용법

* `C`: 연결 시도
* `R <n>`: n번째 파일 요청
* `F`: 연결 종료
